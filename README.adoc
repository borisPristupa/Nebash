= Nebash

== Getting started

.Run app
[source,bash]
----
$ ./gradlew :run
----

.Run tests
[source,bash]
----
$ ./gradlew :test
----

.Generate docs
[source,bash]
----
$ ./gradlew :dokkaHtml
----

.Run static analysis
[source,bash]
----
$ ./gradlew :detekt
----

.Check before commit
[source,bash]
----
$ ./gradlew :check
----
